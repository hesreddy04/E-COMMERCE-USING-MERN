# E-COMMERCE-USING-MERN

This project is an **E-Commerce Web Application** developed using the **MERN stack** (MongoDB, Express.js, React.js, Node.js). The application allows users to browse products, add them to a cart, and proceed with a secure checkout process using payment gateways like **Stripe**.

## Features

- **User Authentication**: Users can sign up, log in, and manage their accounts.
- **Product Listings**: Users can view various products with details, and filter products based on categories.
- **Shopping Cart**: Users can add products to their cart and update the quantity.
- **Secure Payment**: Integrated payment gateway (Stripe) for seamless and secure payment transactions.
- **Responsive UI**: A user-friendly and responsive interface built with **React.js** and **Bootstrap**.
- **Admin Panel**: Admins can manage products, view orders, and track sales.

## Technologies Used

- **Frontend**:
  - React.js
  - Bootstrap (for responsive design)
  - Redux (for state management)

- **Backend**:
  - Node.js
  - Express.js
  - MongoDB (Database)
  - Stripe (Payment Integration)
